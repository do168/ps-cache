# sr-cache
A Node.js module that caching with pub/sub when simultaneous-heavy-request situation 

## Installation 
```sh
npm install sr-cache --save
yarn add sr-cache
bower install sr-cache --save
```
## Usage

### Javascript
```javascript
var srCache = require('sr-cache');
```

### TypeScript
```typescript
import srCache from 'sr-cache';
```
