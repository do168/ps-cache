# ps-cache
A Node.js module that caching with pub/sub when simultaneous-heavy-request situation 

## Installation 
```sh
npm install ps-cache --save
yarn add ps-cache
bower install ps-cache --save
```
## Usage

### Javascript
```javascript
var psCache = require('ps-cache');
```

### TypeScript
```typescript
import psCache from 'ps-cache';
```
